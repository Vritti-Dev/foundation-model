# Module 1. Just Enough NumPy

**Runs in your browser.** Nothing to install.

Every computation in this course is a NumPy array under the hood: every weight, every activation, every gradient. You do not need to be a NumPy expert. You do need a handful of moves to be second nature: creating arrays, indexing and slicing, broadcasting, and the dot product.

This module is a quick, hands-on tour of exactly those operations and nothing more. Each cell has a small task with an inline check, so you know immediately when you have it right. If you already know NumPy, you can blast through this in a few minutes. If you do not, this is the foundation everything else stands on.
