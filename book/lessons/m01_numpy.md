# Module 1 — Just Enough NumPy

**Runs in your browser.** Nothing to install.

Every computation in this course — every weight, activation, and gradient — is a
NumPy array under the hood. You don't need to be a NumPy expert, but you do need
a handful of moves to be second nature: creating arrays, indexing and slicing,
broadcasting, and the dot product.

This module is a quick, hands-on tour of exactly those operations and nothing
more. Each cell has a small task with an inline check, so you'll know
immediately when you've got it right. If you already know NumPy, you can blast
through this in a few minutes; if you don't, this is the foundation everything
else stands on.
